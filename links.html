USEFUL LINKS

A hand-picked list of resources that we used when learning to code and still refer back to frequently.

Introduction to R

http://tryr.codeschool.com introduction to r, vectors, data frames, matrices.
http://blog.revolutionanalytics.com/2012/12/coursera-videos.html Videos from Coursera’s R course.
http://www.r-bloggers.com A metablog of R tutorials
http://blog.revolutionanalytics.com/2012/12/coursera-videos.html Videos from an online course on R (intro to R, data formatting, data visualisation)
http://www.blopig.com/blog/2013/07/citing-r-packages-in-your-thesispaperassignments/ How to cite R and R packages in your assignment/thesis/paper

Graphics and Data Visualisation

susanejohnston.wordpress.com/2013/08/30/base-graphics-in-r-a-detailed-idiots-guide/ Base Graphics in R: A Detailed Idiot’s Guide.
ggplot2.org/book/ Sample chapters from Hadley Wickham’s ggplot2 book.
research.stowers-institute.org/efg/R/Color/Chart/ColorChart.pdf R colour values
ase.tufts.edu/bugs/guide/assets/R%20Graphics%20Cookbook.pdf Graphics guide, mostly ggplot2.
shiny.rstudio.com Shiny tutorials

R Markdown, Version Control & Reproducible Research

susanejohnston.wordpress.com/2015/05/12/an-introduction-to-reproducible-research-in-r-and-r-studio/ An Introduction to Reproducible Research in R and R Studio.
www.latex-project.org LaTeX
help.github.com/articles/good-resources-for-learning-git-and-github/ Github tutorial repository.
github.com/adam-p/markdown-here/wiki/Markdown-Here-Cheatsheet Markdown cheatsheet.
www.rstudio.com/resources/webinars/introducing-notebooks-with-r-markdown/ R notebooks introduction video.
reu.dimacs.rutgers.edu/Symbols.pdf LaTeX math mode syntax.
https://tobi.oetiker.ch/lshort/lshort.pdf Concise guide to LaTeX programming.
britishecologicalsociety.org/wp-content/uploads/Publ_Data-Management-Booklet.pdf British Ecological Society Guide to Data Management in Ecology and Evolution.
github.com/BES2016Workshop Materials for the version control and reproducible code workshops at the British Ecological Society 2016 Annual Meeting.

Statistical Analysis

rcompanion.org/documents/RCompanionBioStatistics.pdf Standard textbook for biological statistics using R.
r4ds.had.co.nz/ Hadley Wickham's R for Data Science book.
www.bodowinter.com/tutorial/bw_LME_tutorial1.pdf Linear models and linear mixed effects models in R: Tutorial by Bodo Winter.
www.bodowinter.com/tutorial/bw_LME_tutorial2.pdf A very basic tutorial for performing linear mixed effects analyses by Bodo Winter.

Data Manipulation

students.brown.edu/seeing-theory/index.html Seeing Theory - a stunning visual guide to statistics.
seananderson.ca/2014/09/13/dplyr-intro.html The dplyr package and pipes.
www.rstudio.com/wp-content/uploads/2015/02/data-wrangling-cheatsheet.pdf Data wrangling cheatsheet, dplyr, tidyr.
www.rstudio.com/resources/cheatsheets/ Cheatsheets for ggplot, dplyr, Shiny, etc.

Shiny

http://shiny.rstudio.com/tutorial/ The official shiny tutorial.
